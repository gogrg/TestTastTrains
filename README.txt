Входные данные задаются в файл в формате JSON
Файл лежит в папке config, которая в корне
Путь к файлу указывается в Main

Что должно быть во входных данных:
stations - список станций с вместимостью
routes - матрица маршрутов. 0 - нет пути
trains - данные о поезде. Его номер и маршрут

Формат входных данных:
stations: ["Название станци : вместимость", "Следующая станция : вместимость"],
routes: ["0 1 2",
         "1 0 3".
         "2 3 0"],
trains: [{"number": номер поезда, "route": маршрут поезда"},..]

Пример входных данных в файле TextInputData1

Маршрут задаётся списком названий станций через запятую

Проверка производится по двум параметрам:
1) На станции: проверка количества одновременно приехавших поездов.
Если их больше, чем вместимость станции - происходит авария
2) На путях: если на одном пути находятся одновременно 2 поезда, сравниваются их направления движения.
Если они движутся в одну сторону, то, так как скорости одинаковые, столкновения не будет.
Если они движутся навстречу - произойдёт авария
